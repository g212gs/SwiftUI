# Features

- [x] Swift UI - regular controlls like List, HStack, VStack, ZStack, Navigation, button click, Shadow, Overlays
- [x] Dark mode handling
- [x] State handling
- [x] Data handling by model
- [x] Spacer, Divider


## Requirements

- iOS 13.0+
- Xcode 11.0+
- Swift 5.0+
